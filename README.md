# First-git

1、首先在github官网新建个文件夹

2、将云端的文件夹克隆到本地

​	git clone https://github.com/Leon199601/First-git.git

3、初次链接

​	cd

​	git init

​	git commit -m “备注”

​	git branch -M main

​	git remote add origin https://github.com/Leon199601/First-git.git

​	git push -u origin main

3、更改文件，eg：更改readme文件

​	cd

​	git init

​	git add *

​	git commit -m "备注"

​	git push
